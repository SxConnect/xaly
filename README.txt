# Xaly

**Xaly - eXperiência Avançada com Linguagens e IA**  
Uma interface de conversação com IA, personalizada para uso multiusuário, white label e integrada à OpenRouter.

---

## ✨ Visão Geral

Xaly é uma interface personalizada com foco em:

- White Label total (ícones, nome e estilo)
- Multiusuário com login e painel de administração
- Conexão com OpenRouter via API Key
- Integração com fluxo de billing (n8n + Baserow)
